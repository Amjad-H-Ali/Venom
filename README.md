# VENOM

### Syntax

##### *Variable Declaration*
```
foo is 9

foo = 9.73

bazz is true

message is "Hello, World!"

```
#####  *Array Definition*

```
bar is [7, "some string", false]
```

#####  *Object Definition*

```
Person is :
  
  name is "John"
  
  age is 27
  
  height is 6.3
  
```
##### *Standard Output* 

```
write "Hi!"

// >>  "HI!"
```



##### *Function Definition*

```
foo is |num1, num2| =>

    output num1+num2
 
```
##### *Function Call*

```
write foo|10, 5|

// >>>  15
```

##### *Loop*

```
{ // Some Code  } * 3

// Executes Block Statement three times
```
